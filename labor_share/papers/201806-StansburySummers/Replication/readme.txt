1. Data files:
The raw data necessary is in the Excel Dataset "Replication Dataset.xls"
The tab "Annual data" should be identical to the Stata file "raw.dta". This data is used for everything except Appendix Figures A1 and A2.
The tab "FigureA1 data" should be identical to the Stata file "compensationsharebypercentile.dta".
The tab "FigureA2 data" should be identical to the Stata file "crosscountrymedianincome.dta".

2. Run "replication_datasetup.do" to convert the raw data "raw.dta" into the final dataset "final.dta"

3. To replicate regressions in text, use "replication_regressions.do".
Note that the table output into Word documents are not fully formatted/labelled.

4. To replicate regressions in the Appendix, use "replication_regressions_appendix.do"

5. To replicate the graphs, use "replication_graphs.do"

6. To replicate Table 6, use the tab "Table6 calculations" in the Excel file "Replication Dataset.xls"

If you wish to replicate the results we list as "available on request" in our paper, please contact us.
